## 七牛图床管理

前端: vue、vuex、axios、iview、vue-router

服务端: expressjs、七牛SDK

[DEMO](http://qim.jakeyu.top)

## 使用

```sh
git clone https://github.com/FeddyTeam/qiniu-images-manager.git
cd qiniu-images-manager
```

### 前端

```sh
npm i
npm run dev
```

### 服务端

```sh
cd server
npm i
npm run dev
```
